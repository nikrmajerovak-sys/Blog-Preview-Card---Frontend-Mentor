# Blog Preview Card Challenge

## Project Description
This project is a solution to the **Frontend Mentor – Blog Preview Card** challenge.  
The goal is to create a **responsive card** with an image, article title, tags, and author information.

## Technologies Used
- HTML5
- CSS3

## Features
- Responsive design – card adapts to mobile screens 
- Hover effects on links and buttons  
- Clean, modern layout following the Frontend Mentor design
